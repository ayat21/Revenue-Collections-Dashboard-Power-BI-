# Revenue & Collections Dashboard (Power BI)

## Overview

This project analyzes telecom data across multiple providers and departments to track:

* Revenue & Collections
* Profitability
* Customer segmentation
* Performance by department and provider

## Key Metrics

* Total Revenue
* Total Collections
* Collection Rate %
* Profit & Profit Margin
* ARPU (Average Revenue Per User)
* Outstanding Amount

## Key Insights

* Identified loss-making departments
* Highlighted free/internal accounts impacting cost
* Compared performance across providers

## Tools Used

* Power BI
* Power Query
* DAX
